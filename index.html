TBGeneral - Minecraft Plugin

TBGeneral is a general-purpose Minecraft plugin designed to enhance your server's functionality. With customizable features, easy-to-use commands, and no coding required, it's perfect for server admins who need practical tools without hassle.

  • Features:
    • Customizable configuration
    • No coding required
    • Easy-to-use commands
    • Minimal performance impact
  • Installation:

    Simply download from GitHub, drop the plugin into your server's "plugins" folder, and modify the configuration file as needed.
